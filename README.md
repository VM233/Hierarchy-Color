# Hierarchy Color

Unity editor extension for coloring Hierarchy items and showing component icons.

## Installation

Add the package through Unity Package Manager with this Git URL:

```text
https://github.com/Atara-Studio/Hierarchy-Color.git
```

Or add it to `Packages/manifest.json`:

```json
"com.atara-studio.hierarchy-color": "https://github.com/Atara-Studio/Hierarchy-Color.git"
```

## Settings

Open `Project Settings > Hierarchy Color` to configure color presets and component icon display.
